JDK安装

开发 Java 需要安装 JDK（Java Development Kit），即 Java 开发工具箱。在Java 1.2 ~ 1.4 时期，被称为 Java SDK（Software Development Kit）。

另外我们还会经常看到 JRE（Java Runtime Environment），即 Java 运行时的环境。

JDK 包含了 JRE，我们不需要重复安装 JRE，对于只需要执行 Java 的设备，可以只安装 JRE。
