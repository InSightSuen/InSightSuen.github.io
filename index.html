总结

修改 Style attributes 时候

  • Parent Style 来自 @android:xxx 时，在属性名前添加 android:
    例如 <item name="android:windowActionBar">false</item>
  • Parent Style 不来自@android:xxx 时，无需在属性名前添加 android:
    例如 <item name="windowActionBar">false</item>

JDK安装

开发 Java 需要安装 JDK（Java Development Kit），即 Java 开发工具箱。在Java 1.2 ~ 1.4 时期，被称为 Java SDK（Software Development Kit）。

另外我们还会经常看到 JRE（Java Runtime Environment），即 Java 运行时的环境。

JDK 包含了 JRE，我们不需要重复安装 JRE，对于只需要执行 Java 的设备，可以只安装 JRE。

ADB

ADB，即 Android Debug Bridge，是 Android 开发中通过 PC 端控制 Android 设备的重要命令行工具。

ADB 是一个客户端-服务端程序，包含以下三部分功能：

  • 一个客户端，运行在开发 PC 上。可以通过 ADB 命令行来调用客户端。其他的 Android 工具，例如 ADT 和 DDMS 都是由 ADB 创建的。
  • 一个服务端，以后台进程运行在开发 PC 上。服务端用于客户端与模拟器或者 Android 设备上的守护进程的通讯。
  • 一个守护进程，以后台进程运行在模拟器或者 Android 设备上。
